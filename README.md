## 🎯 What You'll Build

A **Serverless URL Shortener** — a full-stack web application using:
- **S3** — Host the frontend website
- **Lambda** — Backend logic (no servers!)
- **API Gateway** — REST API endpoints
- **DynamoDB** — Database for storing URLs
- **IAM** — Security and permissions
- **VPC & Security Groups** — Network security for EC2
